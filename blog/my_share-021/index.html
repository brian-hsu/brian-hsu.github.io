Python debug 神器之一

By 捕萊恩 16 June 2021

Never use print for debugging again 真的講得很好~

新手最土炮的方法就是在問題之前 print 函式

有幸可以不再使用 print 真的是很幸福的事

來看看範例

import pysnooper

@pysnooper.snoop()
def dict_for2():
    dict1 = {
        'a': 1,
        'b': 2,
        'c': 3
    }

    for k, v in dict1.items():
        print("key is : {}".format(k))
        print("value is : {}".format(v))
